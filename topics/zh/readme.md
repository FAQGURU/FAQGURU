## 这是什么？

这里收录了一系列超流行的面试问题、答案和代码片段，可以帮助你准备技术面试。这里的许多问题都是真实面试中的真实问题。虽然不会很全面，答案也相对精简（其中的具体概念，网上可能有更好或更深入的文章），你可以当做参考工具使用，来帮助你准备技术面试。

注意：请记住，这里有许多是开放式的问题，从而会引发有趣的讨论，讨论会更多地告诉你个人的能力信息，而不是直接答案。
